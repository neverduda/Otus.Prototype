# Otus.Prototype

Домашнее задание к уроку "Порождающие шаблоны проектирования":

0) Придумать и создать 3-4 класса, которые как минимум дважды наследуются и написать краткое описание текстом.
- Созданы классы BasketballPlayer, FootballPlayer, VolleyballPlayer, которые наследуются от Sportsman<T> и реализуют интерфейсы ICloneable и IMyCloneable<T>. Описание классов внутри классов. 
 
1) Создать свой дженерик интерфейс IMyCloneable для реализации шаблона "Прототип".
- IMyCloneable<T> создан. Он реализуется абстрактным классом Sportsman<T>

2) Сделать возможность клонирования объекта для каждого из этих классов, используя вызовы родительских конструкторов.
- такая возможность реализована для всех классов (BasketballPlayer, FootballPlayer, VolleyballPlayer) в методе MyClone() интерфейса IMyCloneable<T>

3) Составить тесты или написать программу для демонстрации функции клонирования.
- демонстрация реализована в классе Program

4) Добавить к каждому классу реализацию стандартного интерфейса ICloneable и реализовать его функционал через уже созданные методы.
- Каждый класс(BasketballPlayer, FootballPlayer, VolleyballPlayer) имеет реализацию интерфейса ICloneable - метод object Clone()

5) Написать вывод: какие преимущества и недостатки у каждого из интерфейсов: IMyCloneable и ICloneable.
 - написал в интерфейсе для сдачи ДЗ
